# simple-type-alias-test

Test fixture for simple type aliases (not unions or intersections)

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := simple-type-alias-test.New("https://api.example.com")
    // Use the SDK...
}
```
