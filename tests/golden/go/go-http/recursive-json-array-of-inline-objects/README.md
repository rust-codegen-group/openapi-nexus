# array-of-inline-objects-test

Test case for array of inline objects with snake_case to camelCase conversion (main case from user issue)

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := array-of-inline-objects-test.New("https://api.example.com")
    // Use the SDK...
}
```
