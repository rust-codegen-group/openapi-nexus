# deeply-nested-inline-objects-test

Test case for three levels of nested inline objects

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := deeply-nested-inline-objects-test.New("https://api.example.com")
    // Use the SDK...
}
```
