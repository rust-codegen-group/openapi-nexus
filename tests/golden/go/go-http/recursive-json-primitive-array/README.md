# primitive-array-test

Test case for arrays with primitive items (should not be affected by recursive parsing)

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := primitive-array-test.New("https://api.example.com")
    // Use the SDK...
}
```
