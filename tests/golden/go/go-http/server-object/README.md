# test-api-with-server-objects

This is a test API specification that includes various server configurations

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := test-api-with-server-objects.New("https://api.example.com")
    // Use the SDK...
}
```
