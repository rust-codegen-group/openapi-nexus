# nested-object-reference-test

Test case for nested object reference with recursive FromJSON calls

**Version:** 1.0.0

## Installation

```bash
go get example.com/sdk
```

## Usage

```go
package main

import (
    "context"
    "example.com/sdk"
)

func main() {
    sdk := nested-object-reference-test.New("https://api.example.com")
    // Use the SDK...
}
```
